description: 启动 agent-browser 可视化仪表盘（浏览器访问 localhost:4848）
---

请执行以下操作：

1. 运行 `agent-browser dashboard start` 启动可视化仪表盘
2. 启动成功后，输出提示信息：

```
agent-browser 可视化仪表盘已启动，请在浏览器中打开：

    http://localhost:4848/

即可查看实时浏览器会话、截图和操作日志。
```

如果仪表盘已在运行，直接提示访问地址即可。
使用 `agent-browser dashboard stop` 可停止仪表盘。
